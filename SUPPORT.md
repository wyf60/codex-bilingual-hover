# Support

Use [GitHub Issues](https://github.com/wyf60/codex-bilingual-hover/issues) for reproducible bugs and feature requests. Include:

- operating system and version;
- plugin version;
- whether the issue occurs on the plugin directory, a plugin detail page, or a normal task page;
- the exact helper status output;
- a screenshot with private information removed, when useful.

Do not post passwords, authentication tokens, private workspace content, or unredacted personal information.

For macOS permission problems, verify both Accessibility and Screen Recording. A newly signed build may need to be removed and re-added in System Settings.
