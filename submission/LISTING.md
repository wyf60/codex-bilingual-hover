# Public listing copy

## Plugin name

Codex Bilingual Hover

## Short description

Hover over English plugin descriptions in Codex to see a stable Chinese translation.

## Long description

Codex Bilingual Hover helps Chinese-speaking users understand English text in the Codex plugin directory and plugin detail pages. Rest the pointer over a plugin card, title, example, or description for about 0.2 seconds to see Chinese together with the source text. The overlay follows the text length and stays stable while the pointer remains in the same region.

The helper is deliberately restricted to plugin browsing surfaces. It rejects normal tasks, chats, answer content, the composer, and file previews. macOS uses Accessibility APIs, Apple's system translation framework, and an optional on-device Vision OCR fallback. Windows uses UI Automation and a bundled offline dictionary. The plugin does not patch Codex or upload UI text to a developer-controlled translation service.

## Category

Productivity

## Starter prompts

1. Install and start Codex Bilingual Hover on this computer.
2. Check whether the hover translation helper has the required permissions and is running.
3. Stop Codex Bilingual Hover.
4. Enable launch at login for Codex Bilingual Hover.
5. Remove Codex Bilingual Hover and its launch-at-login configuration.
